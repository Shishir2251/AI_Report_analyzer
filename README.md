# AI Report Analyzer


An AI-powered system that analyzes uploaded reports, detects structural issues,
assesses risk levels, auto-corrects formatting, and compares future reports
against a standardized reference.


## Features
- Report structure validation
- Risk classification (Low / Moderate / High)
- Auto-fixed professional report generation
- Report comparison & similarity scoring
